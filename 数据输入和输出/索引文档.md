### 使用自定义的ID
```bash
PUT /{index}/{type}/{id}
{
  "field": "value",
  ...
}
```
```bash
PUT /website/blog/123
{
  "title": "My first blog entry",
  "text":  "Just trying this out...",
  "date":  "2014/01/01"
}
```
### Autogenerating IDs
```bash
POST /website/blog/
{
  "title": "My second blog entry",
  "text":  "Still trying this out...",
  "date":  "2014/01/01"
}
```
