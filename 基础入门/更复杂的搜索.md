```bash
GET /megacorp/employee/_search
{
    "query" : {
        "bool": {
            "must": {
                "match" : {
                    "last_name" : "smith" []
                }
            },
            "filter": {
                "range" : {
                    "age" : { "gt" : 30 } []
                }
            }
        }
    }
}
```
- `range`_过滤器_，它能找到年龄大于30的文档，其中`gt`表示_大于_(_great than_)。
